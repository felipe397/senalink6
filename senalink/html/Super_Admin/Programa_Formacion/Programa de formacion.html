Nombre del programa

Análisis y desarrollo de software


Este programa está diseñado para formar profesionales en el
campo del análisis y desarrollo de software,
con una formación integral en programación y diseño de sistemas.

Habilidades adquiridas:

Desarrollo de aplicaciones web y móviles.

Conocimiento en lenguajes de programación como Java, Python, JavaScript, entre otros.

Diseño y administración de bases de datos.

Capacidad de análisis de requisitos y especificaciones de sistemas.

Uso de herramientas de control de versiones como Git.

Pruebas de software y control de calidad.

Trabajo en equipo y gestión de proyectos ágiles.
